# SOLPI Professional

**SOLPI - Smart Operations Link for GLPI**

## Descrição

SOLPI é um plugin profissional para GLPI que integra:

- GLPI
- Zabbix
- Evolution API
- WhatsApp
- Inteligência Artificial
- Dashboard em tempo real
- Base de Conhecimento

## Objetivo

Automatizar o ciclo completo de tratamento de incidentes:

Zabbix → SOLPI → GLPI → WhatsApp → IA → Encerramento automático

## Tecnologias

- PHP 8.2+
- GLPI 11
- Composer
- Docker
- MariaDB
- REST API
- JavaScript
- Bootstrap
